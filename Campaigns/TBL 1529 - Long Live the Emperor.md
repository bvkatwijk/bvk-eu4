# Preparing Tag Switches

In a few decades (perhaps around 1580) we will want to do a chain of tag switches, so we need to make preparations ahead of time.

We will have to conquer a lot of regions for the various tags that we want to form:
- Piedmont for [[Sardinia-Piedmont]]
- [[Switzerland]]
- [[Armenia]]
- Palestine for [[Jerusalem]]

Our [[Capital]] state contains quite high development, meaning it can be tough to switch to a small primary [[Culture]]. For example, Croatian has less development  in total (59) than our [[Capital]] state (80). So we need to move our [[Capital]]. This gets very expensive quickly (around 700 admin to move next to Praha). Developing Croatian land 21 times is another expensive option.

Forming [[Romania]] first is a solution to this, as they have a mission to relocate the [[Capital]]. The total development of Romanian [[Culture]] is high enough so that we can shift to it. Accepting and shifting will cost 200 diplo.
