# Elective Monarchy

Bohemia has an Elective Monarchy which allows picking a different ruler on ruler death. This makes the 10 IA for the [[Holy Roman Empire|HRE]] trigger twice, so very useful for passing reforms. Also triggers [[Personal Union|PU]] inheritance twice, making free integrations much more likely.
